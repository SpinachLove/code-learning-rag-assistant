#  代码学习智能审查助⼿
基于RAG检索增强⽣成技术与DeepSeek代码⼤模型开发的编程学习辅助系统，⾯向编程初学者，提供代码智能分析、编程
问题精准答疑能⼒，解决学习过程中代码看不懂、报错不会改、知识点不理解的痛点。
## 项⽬核⼼功能
1.  **代码全维度智能分析**：⽀持Python/Java等主流编程语⾔，实现语法纠错、逻辑解释、复杂度分析、优化建议
四⼤核⼼能⼒。
2.  **基于RAG的编程智能答疑**：构建了编程学习知识库，通过两阶段检索优化，实现精准的知识点解答，严格控制⼤
模型幻觉问题，答案可追溯、可验证。
3.  **可视化Web交互界⾯**：基于Streamlit开发的轻量化界⾯，⽀持代码编辑器、多轮对话、历史记录、引⽤来源展
示，开箱即⽤。
## 技术栈选型
| 模块 | 技术选型 | 选型说明 |

| 前端框架 | Streamlit | 纯Python开发，快速实现可视化交互，适配代码输⼊场景 |

## 系统架构设计
本项⽬采⽤分层架构设计，从上到下分为：- ⽤户层：⽤户通过浏览器访问Web界⾯- 前端交互层：Streamlit实现的代码输⼊、对话交互、结果展示界⾯- 接⼝层：FastAPI封装的标准化RESTful接⼝，提供代码分析、RAG检索、智能问答能⼒- 核⼼能⼒层：代码分析模块、RAG检索引擎、智能问答链- 模型层：Embedding模型、重排序模型、DeepSeek代码⼤模型- 数据存储层：ChromaDB向量数据库、编程学习知识库
## 项⽬核⼼
1.  **针对性解决RAG检索精度问题**：设计了「Markdown标题+固定字符」的双层分块策略，保证编程⽂档语义完整
性；采⽤「向量粗召回+Re-rank精排」的两阶段检索⽅案，经测试，问答准确率提升了30%以上。
2.  **系统性解决⼤模型幻觉问题**：从检索机制、Prompt强约束、模型参数优化、检索精度提升四个维度，构建了全
链路的幻觉管控⽅案，将幻觉发⽣率控制在5%以内，保证答疑内容的准确性。
3.  **垂直场景深度适配**：针对编程学习场景，设计了专属的代码分析Prompt和问答规则，输出内容贴合初学者的学
习需求，⽐通⽤⼤模型更适配编程学习场景。
4.  **完整的⼯程化实现**：完成了从本地开发到线上部署的全流程落地，前后端分离设计，模块解耦，具备良好的可扩
展性和可维护性。
